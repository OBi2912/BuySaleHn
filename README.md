# BuySaleHn - Honduras Marketplace Platform

BuySaleHn is a comprehensive marketplace platform designed specifically for buying and selling miscellaneous items in Honduras. The platform connects buyers and sellers across the country, providing a user-friendly interface for discovering, listing, and purchasing goods.

## üìÑ Pages Overview

### 1. **index.html** - Main Marketplace Page
**Location**: `index.html`  
**Purpose**: The main landing page and product browsing interface

#### Key Features:
- **Product Grid Display**: Shows available items in a responsive grid layout
- **Category Filtering**: Filter products by categories (Electronics, Clothing, Home & Garden, Sports, Books, Automotive, Other)
- **Search Functionality**: Real-time search through product listings
- **Shopping Cart**: Add items to cart with quantity management
- **User Authentication**: Login modal for existing users
- **Product Details**: Click any product to view detailed information
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices

#### Interactive Elements:
- Product cards with hover effects
- Category filter buttons
- Search bar with live filtering
- Cart icon with item count
- Login/Register buttons
- Product detail modals

### 2. **sell.html** - Item Listing Page
**Location**: `sell.html`  
**Purpose**: Interface for sellers to list their items for sale

#### Key Features:
- **Multi-step Form**: Guided process for listing items
- **Image Upload**: Support for up to 8 product photos with drag-and-drop
- **Category Selection**: Choose from predefined categories
- **Pricing**: Set prices in Honduran Lempiras (HNL)
- **Location Selection**: Choose from major Honduran cities
- **Contact Methods**: Specify preferred contact method (Phone, WhatsApp, Email)
- **Terms Agreement**: Required acceptance of terms and conditions

#### Form Sections:
1. **Item Information**: Title, category, price, condition, description
2. **Item Photos**: Image upload area with preview
3. **Location & Contact**: Location selection and contact details
4. **Terms & Agreement**: Legal requirements

### 3. **register.html** - User Registration Page
**Location**: `register.html`  
**Purpose**: New user account creation and setup

#### Key Features:
- **Personal Information**: Name, email, phone, date of birth
- **Location Selection**: Choose city from Honduran locations
- **Account Type Selection**: Choose between Buyer, Seller, or Both
- **Password Security**: Password strength indicator and confirmation
- **Communication Preferences**: Email and SMS notification settings
- **Terms & Privacy**: Required acceptance of terms and privacy policy

#### Account Types:
- **Buyer Account**: Shop for items and make purchases
- **Seller Account**: List items for sale and manage store
- **Both**: Full access to buying and selling features

### 4. **checkout.html** - Checkout Process Page
**Location**: `checkout.html`  
**Purpose**: Complete purchase transactions

#### Key Features:
- **Shipping Information**: Customer address and contact details
- **Payment Methods**: Credit/Debit Card, PayPal, Bank Transfer options
- **Order Summary**: Review items, quantities, and pricing
- **Order Totals**: Subtotal, shipping, tax, and final total
- **Secure Checkout**: Lock icon indicating secure transaction

#### Checkout Flow:
1. **Shipping Details**: Name, email, phone, address
2. **Payment Method**: Select and enter payment information
3. **Order Review**: Confirm all details before placing order
4. **Order Confirmation**: Success message with next steps
